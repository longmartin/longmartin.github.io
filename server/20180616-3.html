SSL 证书错误

正常情况下，你是不会遇到 SSL 证书错误的，除非你的 Ruby 安装方式不正确。

如果遇到 SSL 证书问题，你又无法解决，请修改 ~/.gemrc 文，增加 ssl_verify_mode: 0 配置，以便于 RubyGems 可以忽略 SSL 证书错误。
